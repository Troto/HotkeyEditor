How it works

Key
  • Hover — highlights every command bound to it
  • Click — filters the list to that key’s commands
▾Group👁12
  • Click the name — collapse / expand the group
  • Click the 👁 — highlight all the group’s keys
Command
  • Hover — preview its key(s) on the keyboard
  • Click the name — highlight the bound key
  • Click the hotkey — press desired hotkey
  • Right click — unbind hotkey

Browser shortcuts: keys like Ctrl+N are intercepted as a browser shortcut, so trying to enter it directly won't work. Instead, either tap Ctrl/Shift or click Ctrl/Alt/Shift to "arm" the modifier ready for a second input, then press the key.
