Hotkey Editor -Always keep a backup of your hotkeys.

Works with AoE2:DE (build 177723).
QWERTY Dvorak ?

To get started: load the two files for your current hotkey layout/profile.

Hotkey folder: AoE2:DE hotkeys live in your profile folder at

C:\Users\<you>\Games\Age of Empires 2 DE\<Steam ID>\profile

Each profile is two files — <Name>.hkp at the profile root, and <Name>\Base.hkp inside a subfolder of the same name.

The download will provide both files in that structure; extract them and then copy them back to your profile folder (after backing up the profile and deleting it in game if you play on through Steam).
Apply hotkeys by placing the files in your AoE2 profile folder. On Steam, delete the hotkey layout in-game first so Steam removes its cloud copy.

How it works

Key
  • Hover — highlights every command bound to it
  • Click — filters the list to that key’s commands
▾Group👁12
  • Click the name — collapse / expand the group
  • Click the 👁 — highlight all the group’s keys
Command
  • Hover — preview its key(s) on the keyboard
  • Click the name — highlight the bound key
  • Click the hotkey — press desired hotkey
  • Right click — unbind hotkey

Browser shortcuts: keys like Ctrl+N are intercepted as a browser shortcut, so trying to enter it directly won't work. Instead, either tap Ctrl/Shift or click Ctrl/Alt/Shift to "arm" the modifier ready for a second input, then press the key.
